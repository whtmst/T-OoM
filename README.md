# T-OoM v2.0.0 - Анонсер нехватки маны

**Простой и умный анонсер маны для [Turtle WoW](https://turtlecraft.gg/)**

---

## ✨ Возможности

- 🎯 **3 настраиваемых порога** - задайте точные проценты для мало / критически мало / нет маны
- 🎨 **Полное окно настроек** - ползунки, выпадающие списки, всё в одном месте
- 💾 **Персональные настройки** - у каждого персонажа своя конфигурация
- 🌍 **Два языка интерфейса** - русский (по умолчанию) и английский
- 📍 **Перетаскиваемая кнопка миникарты** - свободный режим (Ctrl+перетаскивание) или вращение вокруг миникарты
- 🏰 **Выбор локации** - открытый мир, подземелья/рейды или везде
- 💬 **3 канала чата** - Say, Party, Raid
- ⚡ **Защита от спама** - каждый порог срабатывает только один раз при падении маны
- 📺 **Оповещение на экране** - крупный текст по центру на 3 секунды

---

## 📥 Установка

1. Скачайте: **[Последняя версия](https://github.com/whtmst/T-OoM/archive/refs/heads/main.zip)**
2. Распакуйте папку `T-OoM` в `World of Warcraft\Interface\AddOns\`
3. Перезапустите игру

---

## ⚙️ Настройка

### Slash-команды

| Команда | Описание |
|---------|----------|
| `/toom` | Показать справку |
| `/toom config` | Открыть окно настроек |
| `/toom lang ru` | Переключить на русский |
| `/toom lang en` | Переключить на английский |
| `/toom reset` | Сбросить позицию кнопки миникарты |

### Кнопка миникарты

- **ЛКМ** - Открыть окно настроек
- **Перетаскивание** - Двигать кнопку вокруг миникарты
- **Ctrl + Перетаскивание** - Свободное перемещение в любое место экрана
- **Ctrl + ПКМ** - Сбросить позицию

---

## 🔧 Описание настроек

### Общие
- **Включить T-OoM** - Вкл/выкл аддон
- **Язык** - Язык интерфейса (ru/en)

### Сообщения о мане
Три текстовых поля для каждого порога. Пользовательские сообщения появляются на экране и в чате.

### Пороги маны
Ползунки от 0 до 100%. По умолчанию: 30% / 15% / 5%.

### Канал чата
Куда отправляются оповещения (Say, Party, Raid).

### Режим локации
- **Открытый мир** - Работает только вне подземелий
- **Подземелья/Рейды** - Работает только внутри (включая БГ и арены)
- **Везде** - Работает всегда

# T-OoM v2.0.0 - Out of Mana Announcer

**Simple & smart mana announcer for [Turtle WoW](https://turtlecraft.gg/)**

---

## ✨ Features

- 🎯 **3 custom mana thresholds** - set exact percentages for low / critical / out of mana
- 🎨 **Full GUI settings window** - sliders, dropdowns, everything in one place
- 💾 **Per-character settings** - each character has its own configuration
- 🌍 **Bilingual interface** - Russian (default) & English, switch with one click
- 📍 **Draggable minimap button** - free mode (Ctrl+drag) or orbit around minimap
- 🏰 **Instance mode selector** - open world, dungeons/raids, or always
- 💬 **3 chat channels** - Say, Party, Raid
- ⚡ **No spam protection** - each threshold fires only once per mana drop
- 📺 **On-screen display** - large text overlay with 3-second duration

---

## 📥 Installation

1. Download: **[Latest Release](https://github.com/whtmst/T-OoM/archive/refs/heads/main.zip)**
2. Unpack `T-OoM` folder into `World of Warcraft\Interface\AddOns\`
3. Restart game

---

## ⚙️ Configuration

### Slash Commands

| Command | Description |
|---------|-------------|
| `/toom` | Show help |
| `/toom config` | Open settings window |
| `/toom lang ru` | Switch to Russian |
| `/toom lang en` | Switch to English |
| `/toom reset` | Reset minimap button position |

### Minimap Button

- **Left Click** - Open settings window
- **Drag** - Move button around minimap
- **Ctrl + Drag** - Free movement anywhere on screen
- **Ctrl + Right Click** - Reset position

---

## 🔧 Settings Explained

### General
- **Enable T-OoM** - Turn addon on/off
- **Language** - Interface language (ru/en)

### Mana Messages
Three text fields for each threshold. Custom messages appear on screen and in chat.

### Mana Thresholds
Sliders from 0–100%. Default: 30% / 15% / 5%.

### Chat Channel
Where announcements are sent (Say, Party, Raid).

### Instance Mode
- **Open World** - Works only outside instances
- **Dungeons/Raids** - Works only inside instances (including BGs/arenas)
- **Always** - Works everywhere
